This project is an online analytics dashboard built for a fictional fitness company called ShoFit Club.  
The purpose of the dashboard is to visualize synthetic business data related to gym operations, including:

- Member growth
- Membership distribution
- Class attendance
- Customer satisfaction
- Revenue generation

The dashboard was created using:

- HTML
- Bootstrap 5
- JavaScript
- Chart.js


This project demonstrates responsive layout design, interactive visualizations, data parsing, and real-world business analytics concepts.
